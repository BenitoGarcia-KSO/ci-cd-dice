\# CI/CD Dice



Aplicación Python que simula el lanzamiento de dados utilizando la librería dice.



La aplicación dispone de una función que recibe:



\- amount: número de lanzamientos

\- sides: número de caras del dado



La función devuelve una lista con los resultados obtenidos.



Posteriormente se realizan 5 lanzamientos de un dado de 6 caras, mostrando por pantalla el número de lanzamiento y el resultado obtenido, esperando 5 segundos entre lanzamientos.
